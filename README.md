# Features:
## Kernel:
- x86 kernel
- filesystem

## Libraries:
- runtime support for: libc, libcpp, libobjc, libjs
- ui function: libui

## Userland:
- simple terminal and some apps

# Building:
## building pranaOS:
- Build options:

- ```make pranaOS``` to build iso
- ```make clean``` remove the iso
- ```make qemu``` build iso and run it
